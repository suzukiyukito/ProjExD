## 超高機能電卓(ex02/calc.py)
### 追加機能
- 四則演算:＝を押すことによって、Entryに書かれている四則演算式を計算し、表示させる。
- クリア(CA):Entryに入力されている文字や数字を消す
- sin:Entryに入力されている演算式をsinで計算する
- **2:Entryに入力されている演算式の二乗を取り。表示させる
- ():括弧であり、()の中に表記されている演算式が優先的に計算される
- !:階乗値を取り、その値を返す
