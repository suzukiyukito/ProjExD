# 第6回
## こうかとんシューティング（ex06/shooting_koukaton.py）
### ゲーム概要
- ex06/shooting_koukaton.pyを実行すると，1600x900のスクリーンに草原が描画され，現れる敵を倒していくゲーム
- こうかとんが爆弾や敵と接触するとゲームオーバーで終了する
- こうかとんは前方に卵を発射することができる
- 卵と敵が接触した時敵が消える
- 一定間隔で薬とハートが流れてくる
- 薬を取ると一定時間攻撃の制限がなくなる
- ハートを取ると敵や敵の攻撃が消える
### 操作方法
- 矢印キーでこうかとんを上下左右に移動する
- スペースキーで卵の発射（攻撃）ができる
